# Sudoku Game

## Installation

Clone the GitHub repository
```sh
git clone https://github.com/HericC/Sudoku-Game.git
```

## Development Setup

O jogo roda no terminal, utilizando a função "print" da linguagem Python.

Running

Execute o arquivo "Sudoku.py".
